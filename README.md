# EnhancedAdSkip - 增强版广告跳过插件

## 支持的功能
- AVPlayer 视频广告加速
- WKWebView/UIWebView 广告检测绕过
- 穿山甲SDK广告加速
- 广点通SDK广告加速
- 百度广告SDK支持
- 自动点击跳过按钮

## 编译方法

### 方法1: GitHub Actions (推荐)
1. Fork 此仓库
2. 推送代码自动编译
3. 下载 .deb 文件

### 方法2: 本地 macOS
```bash
brew install theos
make package
```

## 安装
将生成的 .deb 文件安装到 iOS 设备
